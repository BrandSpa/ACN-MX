Export & Import Visual Composer Templates (Saved Templates/My Templates) in few clicks.

== Description ==
Export & Import <strong>Visual Composer</strong> Templates (Saved Templates/My Templates) in few clicks.


This plugin is addon for <a href="http://bit.ly/vc-plugin" target="_blank"><strong>WPBakery Page Builder</strong></a> (formerly Visual Composer).


== Features ==
* <strong>Import/Export templates</strong> in JSON format – take your custom templates with you
* <strong>Single Template Export</strong> – export only those templates you wish

== Compatible With ==
* WPBakery Page Builder (formerly <a href="http://bit.ly/vc-plugin" target="_blank"><strong>Visual Composer</strong></a>) 4.12.x/ 5.1.x and 5.4 (Latest version)

== Installation ==
1. Upload the plugin files to the "/wp-content/plugins/vc-templates-import-export/" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Visual Composer -> Templates Import & Export screen to use the plugin.
